LCL 滤波器设计器

三相并网逆变器工程初设
计算原理和调参方向

1. 额定电流与基值

三相相电流：I = P / (sqrt(3) * Vll * pf)。相电压：Vph = Vll / sqrt(3)。基准阻抗：Zbase = Vph^2 / (P / 3)。

2. 逆变侧电感 Li

先设允许电流纹波 ΔI = I * 纹波比例，再用 Li ≈ Vdc / (6 * fsw * ΔI) 估算。纹波比例越小，Li 越大，动态响应会变慢。

3. 电容 Cf 与电网侧 Lg

Cbase = 1 / (2π * fg * Zbase)，Cf = Cbase * 电容比例。Lg = Li * Lg/Li。Cf 太大会带来基波无功，Lg 太大会增加压降。

4. 谐振与阻尼

fres = 1/(2π) * sqrt((Li + Lg)/(Li * Lg * Cf))。通常希望 fres 高于 10 倍电网频率，并低于约 0.5fsw。Rd 为串联阻尼初值。

这些结果用于初步设计和 sanity check，最终还要结合控制延时、采样频率、阻尼方式、并网标准和仿真结果确认。

通用滤波器设计插件

RC / LC / RLC 初值计算
这个插件怎么算

RC 低通 / 高通

截止频率 fc = 1 / (2πRC)。给定 fc 和 C 时，反算 R = 1 / (2πfcC)；同时用你输入的 R/C 回算实际 fc，方便选标准阻值。

LC 二阶低通

谐振频率 f0 = 1 / (2πsqrt(LC))。给定 f0 和 C 时，反算 L = 1 / ((2πf0)^2 * C)。阻尼电阻近似 Rcrit = 2ζsqrt(L/C)。

串联 RLC 陷波

中心频率 f0 = 1 / (2πsqrt(LC))，品质因数 Q = ω0L / R。给定 Q 后，R = ω0L / Q，带宽 BW ≈ f0 / Q。

使用边界
